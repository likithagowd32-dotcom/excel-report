# 📊 Expense Report Generator

This project is a simple Python-based application that reads expense data from a CSV file, analyzes it, and generates:

* 📈 A bar chart showing expenses by category
* 📄 An Excel report with raw data and summary

---

## 🚀 Features

* Reads data from a CSV file
* Creates a pivot table for category-wise expenses
* Generates a bar chart (`chart.png`)
* Exports results to an Excel file (`report.xlsx`)

---

## 🛠️ Technologies Used

* Python 🐍
* pandas
* matplotlib
* openpyxl
